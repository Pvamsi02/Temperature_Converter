# Temperature_Converter
Temperature Converter application which is used to convert diffrent temperatures

# Key Features
- Conversions of Temperatures from diffrent units.
- Unit Conversions
  1. Celsius(°C)
  2. Fahrenheit(°F)
  3. Kelvin(K)
  4. Rankine(°R)
- Good UI interactions.
